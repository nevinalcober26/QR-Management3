# **App Name**: FloorPlanEditor

## Core Features:

- Modal Popup: Display the floor plan editor in a modal popup when the 'Edit Floor Plan' button is clicked.
- Add Room Tab: A section where the user can see tabs such as: Main Dining Room, Private Room 1, Outdoor Patio and VIP Lounge.
- Table Element Selection: Enable users to select from available table elements such as: Round Table, Square Table, Rectangle Table
- Other Element Selection: Allow users to incorporate walls, doors and plants into the editor.
- Drag and Drop: Implement a drag-and-drop interface for adding and positioning elements on the floor plan.
- Table Property Adjustment: Adjust settings, rotation and delete them.
- Smart Suggestion Tool: An AI-powered tool which makes suggestions based on layout of other elements to suggest number of seats, status and best possible dimensions. (These values can then be overridden by the user).

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to convey a modern and creative feel.
- Background color: Light gray (#F5F5F5) to provide a clean, neutral backdrop.
- Accent color: Muted purple (#9370DB) for interactive elements and highlights.
- Body and headline font: 'Inter', a sans-serif font that will give the tool a modern, neutral look
- Use minimalist icons for table elements, walls, doors, and plants.
- Grid-based layout for the floor plan editor to enable precise placement of elements.
- Subtle transitions when dragging and dropping elements or adjusting properties.
